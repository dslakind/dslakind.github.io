About

I teach AP Computer Science A, Cybersecurity, and Engineering Design at PCTI STEM Academy. I mentor teams in national competitions including picoCTF, NASA HUNCH, and Samsung Solve for Tomorrow. My work centers on helping students apply computer science and engineering to real-world problems through creativity, collaboration, and persistence.

Teaching & Program Highlights

As a Computer Science instructor and Cybersecurity advisor, I design and lead project-based programs that connect classroom learning with authentic technical experiences. These initiatives emphasize collaboration, iterative design, and applied programming.

For Students

How to Start a Project on GitHub
  1. Open the starter repository and click Use this template → Create a new repository.
  2. Name it clearly (e.g., nim-period3-teamA) and set it to public unless told otherwise.
  3. Clone to your computer or open in GitHub Codespaces.
  4. Create a branch for your work: git checkout -b feature/your-name.
  5. Commit early, commit often. Use meaningful commit messages.
  6. Open a Pull Request to merge into main and request reviews from teammates.
Submitting Work
  1. Update your README with team info, goals, and how to run your program.
  2. Tag a release v1.0 before the deadline.
  3. Share the repository link in Canvas per assignment instructions.
